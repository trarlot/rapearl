Card

Le composant Card affiche une carte représentant un album ou une playlist, avec son titre, sa couverture, et le nom de l'artiste associé (récupéré à partir de l'IdUser).

new Card($0: Object, IdUser: number, Titre: string, Cover: string)
Parameters
$0 (Object)
Name Description
$0.IdUser any
$0.Titre any
$0.Cover any
IdUser (number) Identifiant de l'utilisateur associé à l'album/playlist.
Titre (string) Titre de l'album/playlist.
Cover (string) Chemin vers l'image de la couverture de l'album/playlist.
Example
<Card IdUser={1} Titre="Titre de l'album/playlist" Cover="/chemin/vers/cover.jpg" />
